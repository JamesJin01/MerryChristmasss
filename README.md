# 🎄 Gesture Christmas Tree (手势控制圣诞树)

这是一个基于 Three.js 和 MediaPipe Hands 的互动 3D 网页应用。通过摄像头捕捉手势，你可以控制一颗绚丽的 3D 圣诞树，体验合拢、散开和抓取照片的魔法效果。

## 🌟 功能特色

1.  **视觉风格**：采用哑光绿、金属金和圣诞红的经典配色，搭配电影级辉光效果 (Bloom)，营造金碧辉煌的高级感。
2.  **手势交互**：无需鼠标键盘，完全通过 AI 手势识别控制场景。
3.  **动态形态**：
    - **合拢态**：粒子汇聚成旋转的圣诞树。
    - **散开态**：粒子在空间中自由漂浮，如星尘般散落。
4.  **照片回忆**：支持上传照片生成“照片云”，并通过手势抓取查看。

## 🎨 颜色设定

- **哑光绿 (Matte Green)**: 圣诞树主体颜色。
- **金属金 (Metallic Gold)**: 装饰球与顶部的星星。
- **圣诞红 (Christmas Red)**: 礼物方块。

## 🖐️ 操作指南 (交互说明)

请确保允许浏览器访问摄像头。将手举起到摄像头前，距离约 0.5 米到 1 米效果最佳。

### 1. 握拳 (Closed Fist) -> 🎄 合拢态 (Tree Mode)
- **动作**：将手掌完全握紧成拳头。
- **效果**：所有漂浮的元素（彩球、照片、方块）会迅速汇聚，组成一棵旋转的圣诞树。这是应用的初始状态。

### 2. 张开五指 (Open Palm) -> ✨ 散开态 (Scatter Mode)
- **动作**：张开手掌，五指伸直。
- **效果**：圣诞树炸开，变成漫天飞舞的粒子和照片云。
- **旋转视角**：在散开状态下，移动手的位置（上下左右）可以旋转观察视角。

### 3. 捏合/抓取 (Pinch) -> 🖼️ 照片放大 (Zoom Mode)
- **动作**：在散开状态下，做“捏合”动作（拇指与食指捏在一起，或像抓东西一样）。
- **效果**：随机抓取一张照片并放大到屏幕中央。
- **退出放大**：再次握拳可回到合拢态。

## 📸 如何上传照片

1. 点击屏幕右上角的 "Add Photos" 按钮。
2. 选择本地图片文件上传。
3. 上传的照片将自动变成圣诞树的一部分，并在散开时作为“照片云”的一部分漂浮。

## 🛠️ 本地运行

1.  确保已安装 Node.js。
2.  在项目根目录运行安装依赖:
    ```bash
    npm install
    ```
3.  启动开发服务器:
    ```bash
    npm run dev
    ```
4.  在浏览器打开 `http://localhost:5173`。

---
**提示**: 请在光线充足的环境下使用，以确保手势识别的准确性。
